# Pakistani Medical Student Association (PMSA) – Static Site

This is a simple static website suitable for Netlify, GitHub Pages, or any static host.

## Pages
- `/` (Home)
- `/purpose.html`
- `/join.html` (Netlify Forms enabled)

## Deploy to Netlify (quick)
1. Create a new site on Netlify and drag‑and‑drop the `pmsa-site` folder.
2. Ensure "Build command" is **empty** and "Publish directory" is the folder root.
3. Form submissions will appear in Netlify › Forms.

## Customize
- Edit text in the HTML files.
- Colors & spacing: `assets/styles.css`
- Logo: `assets/logo.svg`
